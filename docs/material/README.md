# Material de entrada

Pegá acá todo lo que vayas sacando del Word u otras fuentes. Yo lo leo y lo
convierto en nodos del grafo / registros del repositorio.

## Qué dejar acá

- Texto/notas del Word (datasets, variables, fuentes, descripciones).
- Oficios, links de GEE, referencias.
- Cualquier documento suelto que deba entrar al catálogo.

## Formato

No importa el formato (`.md`, `.txt`, `.pdf`, `.docx`, pegar texto en un `.md`).
Si es texto suelto, un `.md` por tema alcanza. Ej:

- `datasets.md` → lista de datasets con fuente, variables, rango temporal
- `fuentes.md` → INAMHI, GEE, Copernicus, etc.
- `variables-ambientales.md` → precipitación, temperatura, NDVI...

Cuando agregues algo, avisame y lo proceso.
